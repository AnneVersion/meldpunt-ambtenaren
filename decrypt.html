🔐 Meldingen Ontsleutelen

Gebruik je private key om versleutelde meldingen van GitHub Issues te lezen.

Hoe werkt het?
1. Laad je private key (private_key.pem) — deze staat in C:\Users\aspij\.meldpunt-keys\
2. Klik op "Laad van GitHub" om alle versleutelde meldingen op te halen
3. De tool ontsleutelt de gevoelige gegevens (emails, verhalen) lokaal in je browser
4. Niets verlaat je computer — alle decryptie gebeurt client-side
